# 💣 Minesweeper (Сапер) - WPF Desktop Game

![C#](https://img.shields.io/badge/c%23-%23239120.svg?style=for-the-badge&logo=c-sharp&logoColor=white)
![.NET](https://img.shields.io/badge/.NET-5C2D91?style=for-the-badge&logo=.net&logoColor=white)
![WPF](https://img.shields.io/badge/WPF-UI-blue?style=for-the-badge)

Класична логічна гра «Сапер» (Minesweeper), реалізована як десктопний додаток за допомогою **C#** та фреймворку **WPF (Windows Presentation Foundation)**. Проект розроблено з акцентом на об'єктно-орієнтоване програмування та чисту організацію ігрової логіки.

## 🌟 Основний функціонал

*   **Класичний геймплей:** Відкриття клітинок, встановлення прапорців для позначення мін, таймер та лічильник мін.
*   **Рекурсивний алгоритм:** Автоматичне відкриття всіх сусідніх безпечних (порожніх) клітинок при натисканні на клітинку без мін навколо.
*   **Гнучка генерація поля:** Випадкове розміщення мін при старті нової гри. Об'єктна модель дозволяє легко масштабувати розміри ігрової сітки.
*   **Збереження/Завантаження (I/O):** Безпечна робота з файловою системою. Користувач може самостійно обирати шлях для збереження або завантаження стану гри без автоматичного створення прихованих файлів.
*   **Інтерактивний UI:** Динамічне оновлення інтерфейсу за допомогою обробки подій (Code-Behind).

## 🛠 Технологічний стек

*   **Мова програмування:** C#
*   **UI Фреймворк:** WPF (XAML)
*   **Архітектурний підхід:** OOP (виділення сутностей `Board`, `Cell`), Code-Behind для взаємодії інтерфейсу та логіки.
*   **Платформа:** .NET (сумісно з Windows)

## 🏗 Структура проекту

Логіка гри розділена на ключові об'єктно-орієнтовані компоненти:
*   `Board.cs` — відповідає за стан усього ігрового поля, генерацію мін та перевірку умов перемоги/поразки.
*   `Cell.cs` — представляє окрему клітинку (її координати, наявність міни, стан: відкрита/закрита/з прапорцем).
*   `MainWindow.xaml` та `MainWindow.xaml.cs` — графічний інтерфейс та обробка дій користувача (кліки миші).
